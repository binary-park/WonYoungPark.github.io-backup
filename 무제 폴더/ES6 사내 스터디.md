## 리터너과 변수, 상수, 데이터 타입

### 변수와 상수

변수란 값이 언제든지 바뀔 수 있는 값을 의미하고, 상수란 값이 한번 할당되면 변하지 않는 값을 의미한다.

- let : 변수
- const : 상수



초기값이 할당하지 않으면 암시적으로 특별한 값 undefined가 할당된다.
